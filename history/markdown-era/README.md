# Markdown-Era History

This directory preserves historical schemas and examples from the earlier Markdown-based repository layout.

- `v0.2/` preserves the exact schema and example state associated with the Markdown-era `v0.2` specification.
- `v0.3/` preserves the exact schema and example state associated with the final Markdown-era `v0.3` specification.

These files are historical reference material. They are not the canonical working schemas or examples for `main`.

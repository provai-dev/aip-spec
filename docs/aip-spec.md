# Compatibility Notice

This file is preserved on `main` for backlink and path compatibility only.

The AIP repository migrated from the earlier Markdown-based specification layout to an IETF/RFC XML-based layout at `v0.3`.

## What Changed

- Before the migration, the specification was maintained as Markdown.
- Starting with the repository-format migration at `v0.3`, the canonical working draft on `main` is maintained in XML.
- The canonical authoring sources are now:
  - [`draft.xml`](../draft.xml)
  - [`sections/`](../sections/)
  - [`references/`](../references/)

## Historical Markdown Version

The prior Markdown-era repository is preserved on branch `archive/markdown-era`.

- The historical Markdown document corresponding to this path is the archived `v0.3` document:
  - `https://github.com/provai-dev/aip-spec/blob/archive/markdown-era/spec/v0.3/aip-spec.md`
- Earlier Markdown-era content, including `v0.2`, is also preserved in that archived branch.

## Contributor Guidance

Do not edit this file when changing the draft.

On `main`, all normative and editorial draft updates should be made in the XML sources and reviewed through the current repository workflow described in:

- [README.md](../README.md)
- [CONTRIBUTING.md](../CONTRIBUTING.md)
- [docs/MIGRATION-FROM-MARKDOWN.md](MIGRATION-FROM-MARKDOWN.md)

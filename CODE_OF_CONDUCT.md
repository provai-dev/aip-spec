# Code Of Conduct

This project expects professional, evidence-based collaboration.

Participants should:

- focus on technical substance
- challenge ideas directly without personal attacks
- distinguish fact, inference, and preference
- preserve a reviewable record of rationale and decisions
- respect that maintainers may slow work down when compatibility or standards quality requires it

Unacceptable behavior includes harassment, intimidation, bad-faith disruption, repeated hostility, or attempts to force publication without review.

Maintainers may moderate discussions, pause threads, request changes in conduct, or decline participation that undermines the repository's ability to do careful public standards work.

If a more formal policy or reporting process is later adopted, this file should be replaced with that explicit policy.
